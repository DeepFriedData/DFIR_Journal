# DFIR Journal - Registrar System

## Architecture
- **Master Registrar** - Root of trust. Registers and manages domains.
- **Domain Registrar** - Per-organization authority. Manages users, nodes, and engagement cards within a single domain.

## Stack
- Backend: Python (Flask)
- Database: SQLite
- Frontend: Plain HTML/JS
- Infrastructure: Docker
